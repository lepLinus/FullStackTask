# Full Stack Demo App using docker
Demo app to show a simple connection between react frontend, REST api and mysql database

## How to start
```bash
# mysql use 8086 port
# node.js use 8080 port
# react use 3000 port
docker-compose up
# if you want to run in background
# docker-compose up -d
```
## Start only Frontend
```bash
# inside ./react-frontend
npm install
npm start
```
